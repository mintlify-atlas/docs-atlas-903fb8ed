## Welcome to Reforestation Saga

Reforestation Saga is an innovative iOS game that uses eye-blink detection to plant trees and save planets from environmental collapse. Join the **Intergalactic Reforestation Unit** and embark on your mission to restore life across the galaxy.

<Info>
This game requires camera access to detect your eye blinks, which serve as the primary control mechanism for planting trees.
</Info>

## Understanding the Game Concept

In Reforestation Saga, you play as a cadet in an intergalactic organization dedicated to saving planets in crisis. Your mission is to plant trees on rotating planets by blinking your eyes at the right moment. The game combines:

- **Eye-blink detection** as the core control mechanism
- **Rotating planet mechanics** that increase in difficulty
- **Progressive level system** with dynamic objectives
- **Real-time camera processing** using Vision framework

The app initializes with the blink detector as a global environment object:

```swift ReforestationSaga_App.swift
@main
struct ReforestationSaga_App: App {
    @StateObject private var detector = EyeBlinkDetectorVision()
    var body: some Scene {
        WindowGroup {
            ContentView()
                .environmentObject(detector)
        }
    }
}
```

## Starting Your First Mission

After tapping "Start Mission" in the IntroView, you'll enter your first level:

### Level Initialization

Each level is dynamically configured based on difficulty:

```swift GameView.swift
func setupScene() {
    let newTrees = treesForLevel(currentLevel)
    self.treesNeeded = newTrees
    let newScene = GameScene(size: UIScreen.main.bounds.size)
    newScene.scaleMode = .resizeFill
    
    // Configure callbacks
    newScene.onTreeHit = {
        treesRemaining -= 1
        if treesRemaining <= 0 {
            showLevelClearPopup = true
        }
    }
    
    // Set rotation based on level
    let rotationDuration: Double
    let rotateLeft: Bool
    
    switch currentLevel {
    case 1...5:
        rotationDuration = 4
        rotateLeft = false
    case 6...10:
        rotationDuration = 3
        rotateLeft = Bool.random()
    default:
        rotationDuration = Double.random(in: 1.5...4)
        rotateLeft = Bool.random()
    }
    
    newScene.configureLevel(
        treesNeeded: treesNeeded,
        rotationDuration: rotationDuration,
        rotateLeft: rotateLeft
    )
}
```

### Trees Required Per Level

The number of trees you need to plant increases progressively:

```swift GameView.swift
func treesForLevel(_ level: Int) -> Int {
    let base = 3
    let cycle = (level - 1) / 5
    let positionInCycle = (level - 1) % 5
    return base + cycle + positionInCycle
}
```

## Basic Gameplay Controls

<Note>
**The only control is blinking your eyes.** No buttons, no taps, just natural eye blinks.
</Note>

### How Blink Detection Works

The game uses Vision framework to detect eye blinks in real-time through the front camera:

```swift GameView.swift
.onChange(of: detector.blinkCount) {
    if showInitialBlinkAlert {
        showInitialBlinkAlert = false
        hasPlantedFirstTree = true
        if (treesRemaining > 0) {
            scene?.shootNeedle()
        }
    } else if !showMissionFailedPopup && !showLevelClearPopup {
        scene?.shootNeedle()
    }
}
```

### First Blink Tutorial

When you start your first level, you'll see a blink alert overlay:

## Understanding Level Objectives

### HUD Elements

The game displays key information at the top of the screen:

```swift GameView.swift
VStack {
    HStack {
        // Left: Trees remaining progress bar
        BarView(
            treesRemaining: treesRemaining,
            totalPollution: treesNeeded)

        Spacer()

        // Right: Current level and sound toggle
        LevelSoundView(
            isMuted: $isMuted,
            currentLevel: currentLevel,
            scene: scene
        )
    }
    .padding(.horizontal, 20)
    
    Spacer()
}
```

<Accordion title="What each HUD element means">
  - **Trees Remaining Bar (Left)**: Shows how many more trees you need to plant to complete the level
  - **Level Number (Right)**: Displays your current level
  - **Sound Toggle (Right)**: Mute/unmute game music and sound effects
</Accordion>

### Win Condition

To complete a level, plant all required trees in the green zones:

```swift GameView.swift
newScene.onTreeHit = {
    treesRemaining -= 1
    if treesRemaining <= 0 {
        showLevelClearPopup = true
        
        DispatchQueue.main.asyncAfter(deadline: .now() + 1.2) {
            showLevelClearPopup = false
            currentLevel += 1
            
            if currentLevel - 1 > gameData.highestLevel {
                gameData.highestLevel = currentLevel - 1
                UserDefaults.standard.savedHighestLevel = gameData.highestLevel
            }
            setupScene()
        }
    }
}
```

### Fail Conditions

You'll fail the mission if your needle hits:

1. **Blue water zones** (ocean areas)
2. **Existing trees** (areas already reforested)

```swift GameView.swift
newScene.onFailZoneHit = {
    showMissionFailedPopup = true
}
newScene.onTreeZoneHit = {
    reason = "pohon"
    showMissionFailedPopup = true
}
```

<Warning>
When you fail a mission, you'll restart from Level 1, but your highest level record is preserved!
</Warning>

### Advanced Techniques

<Accordion title="Adapting to Random Rotations (Levels 6+)">
At level 6, the planet's rotation direction becomes random. Quick tips:

- Observe the direction for 1-2 rotations before shooting
- Green zones pass more frequently with 3-second rotation
- Stay patient and wait for optimal alignment
</Accordion>

<Accordion title="Handling Variable Speed (Levels 11+)">
From level 11 onwards, rotation duration randomizes between 1.5-4 seconds:

```swift GameView.swift
default:
    rotationDuration = Double.random(in: 1.5...4)
    rotateLeft = Bool.random()
```

- Fast rotations (1.5s): Blink quickly when zones appear
- Slow rotations (4s): More time to aim, but requires patience
- Adapt your timing to each level's unique speed
</Accordion>

<Accordion title="Maximizing Your High Score">
To achieve the highest levels:

1. **Start strong**: Clear levels 1-5 quickly to build momentum
2. **Stay focused**: One mistake sends you back to level 1
3. **Rest between sessions**: Tired eyes = less accurate blinks
4. **Perfect your timing**: Aim for the center of green zones
5. **Stay calm**: Rushed blinks often miss the target
</Accordion>
