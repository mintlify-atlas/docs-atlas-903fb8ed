## Overview

Reforestation Saga implements eye blink detection using Apple's Vision framework to enable hands-free gameplay. The system captures video frames, detects facial landmarks, calculates eye aspect ratios (EAR), and triggers game actions based on blink events.

## Core Components

## Camera Switching

The system supports dynamic camera switching between front and back cameras during runtime.

```swift EyeBlinkDetectorVision.swift:152-171
@objc func switchCameraTapped() {
    session.beginConfiguration()
    session.inputs.forEach { session.removeInput($0) }

    currentCameraPosition =
        (currentCameraPosition == .back) ? .front : .back

    guard let newDevice = getCamera(for: currentCameraPosition),
        let newInput = try? AVCaptureDeviceInput(device: newDevice)
    else {
        session.commitConfiguration()
        return
    }

    if session.canAddInput(newInput) {
        session.addInput(newInput)
    }

    session.commitConfiguration()
}
```

<Note>
The camera position affects the image orientation used in Vision requests. Front camera uses `.leftMirrored` while back camera uses `.right`.
</Note>

### Eye Aspect Ratio (EAR) Calculation

The EAR algorithm measures eye openness by comparing vertical and horizontal eye distances.

```swift EyeBlinkDetectorVision.swift:292-309
private func computeEyeAspectRatio(_ eye: VNFaceLandmarkRegion2D)
    -> CGFloat
{
    guard eye.pointCount >= 6 else { return 0 }

    let p2 = eye.normalizedPoints[1]
    let p6 = eye.normalizedPoints[5]
    let p3 = eye.normalizedPoints[2]
    let p5 = eye.normalizedPoints[4]
    let p1 = eye.normalizedPoints[0]
    let p4 = eye.normalizedPoints[3]

    let vertical1 = hypot(p2.x - p6.x, p2.y - p6.y)
    let vertical2 = hypot(p3.x - p5.x, p3.y - p5.y)
    let horizontal = hypot(p1.x - p4.x, p1.y - p4.y)

    return (vertical1 + vertical2) / (2.0 * horizontal)
}
```

<Accordion title="Understanding the EAR Formula">
  The Eye Aspect Ratio formula calculates:
  
  **EAR = (vertical1 + vertical2) / (2 × horizontal)**
  
  - **vertical1**: Distance between top and bottom of outer eye corner
  - **vertical2**: Distance between top and bottom of inner eye corner
  - **horizontal**: Distance between left and right eye corners
  
  When the eye is open, the ratio is higher (~0.2-0.3). When closed, it drops significantly (~0.05-0.1).
</Accordion>

## Blink State Management

The detector uses state tracking to register complete blinks (eye close → eye open transitions).

```swift EyeBlinkDetectorVision.swift:201-223
if let landmarks = face.landmarks,
    let leftEye = landmarks.leftEye,
    let rightEye = landmarks.rightEye
{

    let leftEAR = self.computeEyeAspectRatio(leftEye)
    let rightEAR = self.computeEyeAspectRatio(rightEye)

    if leftEAR < 0.1  && rightEAR < 0.1 {
        if !self.isEyeClosed {
            self.isEyeClosed = true
        }
    } else {
        if self.isEyeClosed {
            self.blinkCount += 1
            DispatchQueue.main.async {
                self.blinkCountLabel?.text =
                    "Blink: \(self.blinkCount)"
            }
            self.isEyeClosed = false
        }
    }
```

<Info>
The threshold value of **0.1** for EAR was empirically determined to reliably detect blinks while avoiding false positives from partial eye closure or squinting.
</Info>

## Visual Feedback

The system draws real-time overlays on detected eye landmarks for debugging and visual feedback.

```swift EyeBlinkDetectorVision.swift:253-280
private func drawLandmark(
    _ landmark: VNFaceLandmarkRegion2D, for face: VNFaceObservation,
    in layer: AVCaptureVideoPreviewLayer
) -> CAShapeLayer {
    let path = UIBezierPath()
    let points = landmark.normalizedPoints

    for i in 0..<points.count {
        let point = points[i]
        let converted = VNImagePointForFaceLandmarkPoint(
            point, face.boundingBox, Int(layer.bounds.width),
            Int(layer.bounds.height))
        if i == 0 {
            path.move(to: converted)
        } else {
            path.addLine(to: converted)
        }
    }
    path.close()

    let shapeLayer = CAShapeLayer()
    shapeLayer.path = path.cgPath
    shapeLayer.strokeColor = UIColor(Color.gold).cgColor
    shapeLayer.lineWidth = 2
    shapeLayer.fillColor = UIColor.clear.cgColor

    return shapeLayer
}
```
