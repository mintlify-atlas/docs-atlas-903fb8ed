## Overview

Reforestation Saga uses SwiftUI components overlaid on top of the SpriteKit game scene to provide player feedback, level information, and interactive controls.

## BarView Component

The pollution indicator bar shows remaining trees and mission progress.

```swift BarView.swift
struct BarView: View {
    var treesRemaining: Int
    var totalPollution: Int
    
    var body: some View {
        VStack(alignment: .leading, spacing: 0) {
            Text("Pollution Indicator")
                .font(Font.custom("Gugi", size: 11))
                .multilineTextAlignment(.center)
                .foregroundColor(Color(red: 0.94, green: 0.84, blue: 0.65))
                .padding(.bottom, 8)
            
            ZStack {
                Image("pollution")
                    .resizable(capInsets: EdgeInsets(top: 0, leading: 8, bottom: 0, trailing: 8), resizingMode: .stretch)
                    .frame(width: CGFloat(16 * totalPollution), height: 31)
                
                HStack(spacing: 2) {
                    ForEach((0..<totalPollution).reversed(), id: \.self) { index in
                        Rectangle()
                            .frame(width: 11.94, height: 19.52)
                            .foregroundColor(.clear)
                            .background(
                                index < (totalPollution - treesRemaining) ?
                                Color(red: 0.99, green: 0.56, blue: 0.13).opacity(0.2) : // cleaned
                                Color(red: 1, green: 0.18, blue: 0.18) // still polluted
                            )
                            .cornerRadius(1.58)
                    }
                }
            }
            
            Text("Trees Remaining: \(treesRemaining)")
                .font(Font.custom("Gugi", size: 11))
                .multilineTextAlignment(.center)
                .foregroundColor(Color(red: 0.94, green: 0.84, blue: 0.65))
                .padding(.top, 12)
        }
    }
}
```

<Steps>
  <Step title="Header Label">
    Displays "Pollution Indicator" title
  </Step>
  
  <Step title="Background Image">
    Stretchable pollution bar image scales with total trees needed
  </Step>
  
  <Step title="Pollution Bars">
    Individual rectangles change color as trees are planted
  </Step>
  
  <Step title="Counter Text">
    Shows exact number of trees remaining
  </Step>
</Steps>

### Dynamic Bar Sizing

The pollution bar width adjusts based on the total number of trees needed.

```swift BarView.swift
Image("pollution")
    .resizable(capInsets: EdgeInsets(top: 0, leading: 8, bottom: 0, trailing: 8), resizingMode: .stretch)
    .frame(width: CGFloat(16 * totalPollution), height: 31)
```

<Info>
Each tree requires 16 points of width, so a level with 5 trees creates an 80-point wide bar.
</Info>

### Color Logic

Rectangles change color from red (polluted) to orange (cleaned) as trees are planted.

```swift BarView.swift
.background(
    index < (totalPollution - treesRemaining) ?
    Color(red: 0.99, green: 0.56, blue: 0.13).opacity(0.2) : // cleaned
    Color(red: 1, green: 0.18, blue: 0.18) // still polluted
)
```

## LevelSoundView Component

Displays current level, highest level, and sound toggle in the top-right corner.

```swift LevelSoundView.swift
struct LevelSoundView: View {
    @Binding var isMuted: Bool
    @EnvironmentObject var gameData: GameData
    var currentLevel: Int
    var scene: GameScene?
    
    var body: some View {
        VStack(alignment: .trailing, spacing: 0) {
            Text("Highest Level: \(gameData.highestLevel)")
                .font(Font.custom("Gugi", size: 11))
                .multilineTextAlignment(.center)
                .foregroundColor(Color(red: 0.94, green: 0.84, blue: 0.65))
                .padding(.bottom, 8)
           
            ZStack {
                Image("box")
                    .resizable()
                    .frame(width: 100, height: 40)
                Text("LEVEL \(currentLevel)")
                    .font(Font.custom("Gugi", size: 14))
                    .multilineTextAlignment(.center)
                    .foregroundColor(Color(red: 0.94, green: 0.84, blue: 0.65))
                    .padding(.bottom, 8)
            }
            .offset(x: 4)
            
            Button {
                isMuted.toggle()
                if isMuted {
                    GameMusicManager.shared.pauseMusic()
                } else {
                    GameMusicManager.shared.resumeMusic()
                }
            } label: {
                Image(isMuted ? "soundoff" :"sound")
                    .resizable()
                    .aspectRatio(contentMode: .fit)
                    .frame(width: 35, height: 35)
                    .padding(.top, -6)
                    .offset(x: 4)
            }
        }
    }
}
```

### Sound Control

The sound button toggles between muted and unmuted states, controlling background music.

```swift LevelSoundView.swift
Button {
    isMuted.toggle()
    if isMuted {
        GameMusicManager.shared.pauseMusic()
    } else {
        GameMusicManager.shared.resumeMusic()
    }
} label: {
    Image(isMuted ? "soundoff" :"sound")
        .resizable()
        .aspectRatio(contentMode: .fit)
        .frame(width: 35, height: 35)
}
```

<Tip>
The button image switches between "sound" and "soundoff" assets based on the muted state.
</Tip>

## UI Layout in GameView

The UI components are positioned using SwiftUI overlays on the game scene.

```swift GameView.swift
VStack {
    HStack {
        // Top-left corner
        BarView(
            treesRemaining: treesRemaining,

        Spacer()

        // Top-right corner

            isMuted: $isMuted,
            currentLevel: currentLevel,

        )
        .offset(y: 8)

    .padding(.horizontal, 20)

```

## Mission Failed Popup

Displays failure information and retry button when the player hits ocean or another tree.

```swift MissionFailedPopUp.swift
struct MissionFailedPopUp: View {
    @EnvironmentObject var gameData: GameData
    @Binding var currentLevel: Int
    @Binding var reason: String
    var isNewHighScore: Bool
    var onNextMission: () -> Void
    
    var body: some View {
        VStack {
            ZStack {
                Image("missionfailedbox")
                    .resizable()
                    .frame(width: 329, height: 160)

                Text(reason == "laut" ? "The Tree floats away..." : "Careful! Tree Crash!")
                    .font(Font.custom("Gugi", size: 26))
                    .multilineTextAlignment(.center)
                    .foregroundColor(Color(red: 1, green: 0.42, blue: 0.36))
                    .frame(width: 306, alignment: .top)
                    .padding(.top, 15 + (-40))
                    .padding(.bottom, 47)

                Text("MISSION FAILED")
                    .font(Font.custom("Genos", size: 20.58992).weight(.regular))
                    .multilineTextAlignment(.center)
                    .foregroundColor(Color(red: 1, green: 0.42, blue: 0.36))
                    .padding(.top, 75 + (-40))
                    .padding(.bottom, 37)

                if isNewHighScore {
                    Text("New Highest Level: \(gameData.highestLevel)")
                        .font(Font.custom("Genos", size: 20.58992).weight(.medium))
                        .multilineTextAlignment(.center)
                        .foregroundColor(Color(red: 0.94, green: 0.84, blue: 0.65))
                        .padding(.top, 140 + (-35))
                        .padding(.bottom, 37)
                } else {
                    Text("Your Level: \(currentLevel - 1)")
                        .font(Font.custom("Genos", size: 20.58992).weight(.medium))
                        .multilineTextAlignment(.center)
                        .foregroundColor(Color(red: 0.94, green: 0.84, blue: 0.65))
                        .padding(.top, 140 + (-35))
                        .padding(.bottom, 37)
                }
            }

            Button(action: { onNextMission() }) {
                ZStack {
                    Image("buttonretry")
                    Text("TRY AGAIN")
                        .font(Font.custom("Gugi", size: 25.74193))
                        .multilineTextAlignment(.center)
                        .foregroundColor(Color(red: 0.02, green: 0.02, blue: 0.18))
                        .padding(.bottom, 14)
                }
            }
            .padding(.top, 20)
        }
    }
}
```

## Z-Index Layering

Components are layered using `zIndex` to ensure proper overlap.

| Level Complete | 1 | Above scene |

## Best Practices

- Use `@Binding` for mutable state passed to child components
- Apply consistent spacing (0, 8, 12, 20 points) across components
- Use custom fonts throughout for unified visual identity
- Layer popups with appropriate `zIndex` values
- Animate state changes with easeInOut for smooth transitions
- Provide clear visual feedback for all player actions
- Use contrasting colors for different states (polluted vs cleaned)
- Include both new high score and current level messages for player motivation
