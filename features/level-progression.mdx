## Overview

Reforestation Saga features an infinite progression system where difficulty increases through more trees required, faster planet rotation, and unpredictable rotation patterns.

## Trees Per Level Algorithm

The `treesForLevel` function calculates how many trees players must plant at each level using a cycle-based formula.

### Core Algorithm

```swift ~/workspace/source/ReforestationSaga/Game/GameView.swift
func treesForLevel(_ level: Int) -> Int {
    let base = 3
    let cycle = (level - 1) / 5
    let positionInCycle = (level - 1) % 5
    return base + cycle + positionInCycle
}
```

### Algorithm Breakdown

<Steps>
  <Step title="Base Value">
    Every level starts with a base of **3 trees**.
  </Step>

  <Step title="Cycle Calculation">
    Divide the level by 5 to determine which "cycle" the player is in.
    - Levels 1-5: Cycle 0
    - Levels 6-10: Cycle 1
    - Levels 11-15: Cycle 2
    - And so on...
  </Step>

  <Step title="Position in Cycle">
    Use modulo 5 to find the position within the current cycle (0-4).
  </Step>

  <Step title="Final Calculation">
    Add base + cycle + position to get the total trees required.
  </Step>
</Steps>

## Rotation Difficulty Scaling

Rotation speed and direction change based on the current level.

```swift ~/workspace/source/ReforestationSaga/Game/GameView.swift
let rotationDuration: Double
let rotateLeft: Bool

switch currentLevel {
case 1...5:
    rotationDuration = 4
    rotateLeft = false
case 6...10:
    rotationDuration = 3
    rotateLeft = Bool.random()
default:
    rotationDuration = Double.random(in: 1.5...4)
    rotateLeft = Bool.random()
}
```

## Level Configuration

Each level is configured with specific parameters when initialized.

```swift ~/workspace/source/ReforestationSaga/Game/GameView.swift
func setupScene() {
    if let oldScene = scene {
        oldScene.removeAllActions()
        oldScene.removeAllChildren()
        oldScene.removeFromParent()
    }
    
    reason = "laut"
    let newTrees = treesForLevel(currentLevel)
    self.treesNeeded = newTrees
    let newScene = GameScene(size: UIScreen.main.bounds.size)
    newScene.scaleMode = .resizeFill
    
    // ... setup callbacks ...
    
    let rotationDuration: Double
    let rotateLeft: Bool

    switch currentLevel {
    case 1...5:
        rotationDuration = 4
        rotateLeft = false
    case 6...10:
        rotationDuration = 3
        rotateLeft = Bool.random()
    default:
        rotationDuration = Double.random(in: 1.5...4)
        rotateLeft = Bool.random()
    }

    newScene.configureLevel(
        treesNeeded: treesNeeded,
        rotationDuration: rotationDuration,
        rotateLeft: rotateLeft
    )

    self.treesRemaining = treesNeeded
    self.scene = newScene
    self.sceneID = UUID()
    resumeGame()
}
```

<Tip>
  The `sceneID` is updated with a new UUID to force SwiftUI to recreate the SpriteView, ensuring clean level transitions.
</Tip>

## High Score Tracking

The game tracks the highest level reached using UserDefaults for persistence.

### GameData ObservableObject

```swift ~/workspace/source/ReforestationSaga/ContentView.swift
class GameData: ObservableObject {
    @Published var highestLevel: Int = UserDefaults.standard.savedHighestLevel
}
```

### UserDefaults Extension

```swift ~/workspace/source/ReforestationSaga/Game/GameView.swift
extension UserDefaults {
    private enum Keys {
        static let highestLevel = "highestLevel"
    }

    var savedHighestLevel: Int {
        get { integer(forKey: Keys.highestLevel) }
        set { set(newValue, forKey: Keys.highestLevel) }
    }
}
```

### Updating High Score

```swift ~/workspace/source/ReforestationSaga/Game/GameView.swift
newScene.onTreeHit = {
    treesRemaining -= 1
    if treesRemaining <= 0 {
        showLevelClearPopup = true

        DispatchQueue.main.asyncAfter(deadline: .now() + 1.2) {
            showLevelClearPopup = false
            currentLevel += 1

            if currentLevel - 1 > gameData.highestLevel {
                gameData.highestLevel = currentLevel - 1
                UserDefaults.standard.savedHighestLevel = gameData.highestLevel
            }
            setupScene()
        }
    }
}
```

<Accordion title="Why currentLevel - 1?">
  When completing level X, `currentLevel` is incremented to X+1 for the next level. The high score represents the highest **completed** level, so we save `currentLevel - 1`.

  Example:
  - Player completes level 5
  - `currentLevel` becomes 6 (for next level)
  - High score saved as 5 (completed level)
</Accordion>

## Level Reset on Failure

When players fail, they return to level 1 but keep their high score.

```swift ~/workspace/source/ReforestationSaga/Game/GameView.swift
if showMissionFailedPopup {
    MissionFailedPopUp(
        currentLevel: $currentLevel,
        reason: $reason,
        isNewHighScore: currentLevel - 1 >= gameData.highestLevel,
        onNextMission: {
            currentLevel = 1
            gameData.highestLevel = UserDefaults.standard.savedHighestLevel
            showMissionFailedPopup = false
            setupScene()
            showInitialBlinkAlert = true
        }
    )
}
```

<Warning>
  The level is reset to **1** on failure, not to a checkpoint. This creates a high-stakes roguelike experience where mistakes are costly.
</Warning>

## Scene-Level Configuration

The GameScene accepts level configuration through its `configureLevel` method.

```swift ~/workspace/source/ReforestationSaga/Game/GameScene.swift
func configureLevel(
    treesNeeded: Int,
    rotationDuration: Double,
    rotateLeft: Bool
) {
    treesShot = 0
    treesTarget = treesNeeded
    resetNeedles()
    setupTreeIndicator()
    treeIndicator?.isHidden = false

    circle.removeAllActions()

    let angle: CGFloat = rotateLeft ? -.pi : .pi
    let rotate = SKAction.rotate(byAngle: angle, duration: rotationDuration)
    let forever = SKAction.repeatForever(rotate)
    circle.run(forever)
}
```

### Parameter Effects

<CardGroup cols={2}>
  <Card title="treesNeeded" icon="hashtag">
    Sets the `treesTarget` for level completion tracking.
  </Card>
  <Card title="rotationDuration" icon="clock">
    Controls how fast the planet rotates (seconds per full rotation).
  </Card>
  <Card title="rotateLeft" icon="rotate-left">
    Determines rotation direction (true = counterclockwise, false = clockwise).
  </Card>
  <Card title="resetNeedles()" icon="broom">
    Clears all previously planted trees from the planet.
  </Card>
</CardGroup>

## Infinite Progression

The game has no maximum level - difficulty continues to scale indefinitely.

```swift ~/workspace/source/ReforestationSaga/Game/GameView.swift
default:
    rotationDuration = Double.random(in: 1.5...4)
    rotateLeft = Bool.random()
```

For levels 11 and beyond:
- Trees required follows the cycle formula (grows infinitely)
- Rotation speed is random between 1.5-4 seconds
- Rotation direction is random
- No upper limit on progression

<Tip>
  The combination of increasing tree requirements and random rotation creates exponentially increasing difficulty, making high-level runs extremely challenging.
</Tip>
