## Overview

Reforestation Saga uses a singleton `GameMusicManager` class built on AVFoundation to manage background music and sound effects throughout the game.

## GameMusicManager Architecture

The audio system uses a singleton pattern to ensure consistent audio management across all game views.

```swift ~/workspace/source/ReforestationSaga/GameMusicManager.swift
class GameMusicManager {
    static let shared = GameMusicManager()
    private var audioPlayer: AVAudioPlayer?
    private var effectPlayers: [AVAudioPlayer] = []

    private init() {}
}
```

<Info>
  The singleton pattern (`static let shared`) ensures only one instance of GameMusicManager exists, preventing audio conflicts and simplifying access from anywhere in the app.
</Info>

## Background Music

The background music system supports looping audio tracks during gameplay.

### Playing Music

```swift ~/workspace/source/ReforestationSaga/GameMusicManager.swift
func playMusic(filename: String = "GameEasy", fileExtension: String = "mp3") {
    guard
        let url = Bundle.main.url(
            forResource: filename, withExtension: fileExtension)
    else {
        print("🔴 Failed to find music file")
        return
    }

    do {
        audioPlayer = try AVAudioPlayer(contentsOf: url)
        audioPlayer?.numberOfLoops = -1  // loop forever
        audioPlayer?.prepareToPlay()
        audioPlayer?.play()
    } catch {
        print("🔴 Failed to play music: \(error)")
    }
}
```

<Accordion title="Understanding numberOfLoops = -1">
  Setting `numberOfLoops` to `-1` makes the audio player loop indefinitely. Other values:
  - `0`: Play once (no loop)
  - `1`: Play twice (loop once)
  - `2`: Play three times (loop twice)
  - `-1`: Loop forever
</Accordion>

## Sound Effects

Sound effects are handled separately from background music, allowing multiple effects to play simultaneously.

### Playing Sound Effects

```swift ~/workspace/source/ReforestationSaga/GameMusicManager.swift
func playSoundEffect(filename: String, fileExtension: String = "mp3") {
    guard
        let url = Bundle.main.url(
            forResource: filename, withExtension: fileExtension)
    else { return }

    do {
        let player = try AVAudioPlayer(contentsOf: url)
        player.volume = 1.0
        player.prepareToPlay()
        player.play()
        effectPlayers.append(player)
    } catch {
        print("Error playing sound effect: \(error)")
    }

    // Clean up finished players
    effectPlayers.removeAll { !$0.isPlaying }
}
```

<Note>
  Each sound effect gets its own `AVAudioPlayer` instance, allowing multiple sounds to overlap. Finished players are automatically cleaned up to prevent memory leaks.
</Note>

## Music State Management

Music playback is automatically paused and resumed based on game state.

### Pause on Popup

```swift ~/workspace/source/ReforestationSaga/Game/GameView.swift
.onChange(of: showMissionFailedPopup) { newValue in
    if newValue {
        pauseGame()
        let soundEffectName: String = reason == "laut" ? "water" : "failed"
        GameMusicManager.shared.playSoundEffect(filename: soundEffectName)
        GameMusicManager.shared.pauseMusic()
    } else {
        resumeGame()
        GameMusicManager.shared.resumeMusic()
    }
}
```

<Accordion title="Why pause music on popup?">
  When failure or success popups appear, the game pauses. Background music is also paused to:
  - Reduce audio distraction during important UI moments
  - Allow sound effects to be clearly heard
  - Conserve battery during idle moments
  - Create a natural "breathing space" between attempts
</Accordion>

## Audio Player Management

### Effect Player Cleanup

The manager automatically cleans up finished sound effect players to prevent memory buildup.

```swift ~/workspace/source/ReforestationSaga/GameMusicManager.swift
effectPlayers.removeAll { !$0.isPlaying }
```

<Warning>
  Without cleanup, each sound effect would create a new AVAudioPlayer that persists in memory even after finishing. The `removeAll` call prevents memory leaks.
</Warning>

### Music Player Lifecycle

```swift ~/workspace/source/ReforestationSaga/GameMusicManager.swift
func stopMusic() {
    audioPlayer?.stop()
    audioPlayer = nil
}
```

<Tip>
  Setting `audioPlayer = nil` after stopping releases the AVAudioPlayer instance, freeing up memory and system resources.
</Tip>

</Info>
